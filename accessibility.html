Accessibility Statement

Our commitment to making FlowTrack accessible to everyone

Our Commitment to Accessibility

FlowTrack is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying the relevant accessibility standards to guarantee we provide equal access to all users.

Conformance Status

The Web Content Accessibility Guidelines (WCAG) defines requirements for designers and developers to improve accessibility for people with disabilities. It defines three levels of conformance: Level A, Level AA, and Level AAA.

FlowTrack is fully conformant with WCAG 2.2 Level AA. Fully conformant means that the content fully conforms to the accessibility standard without any exceptions.

Accessibility Measures

FlowTrack takes the following measures to ensure accessibility of our website:

  • Include accessibility as part of our mission statement
  • Include accessibility throughout our internal policies
  • Integrate accessibility into our procurement practices
  • Provide continual accessibility training for our staff
  • Assign clear accessibility goals and responsibilities
  • Employ formal accessibility quality assurance methods

Accessibility Features

Our website includes the following accessibility features:

Perceivable

  • Text alternatives: All non-text content has text alternatives (alt text for images, labels for form controls)
  • Captions and alternatives: Time-based media includes appropriate alternatives
  • Adaptable content: Information is presented in different ways without losing meaning
  • Color contrast: All text meets WCAG 2.2 AA contrast requirements (minimum 4.5:1 for normal text, 3:1 for large text)
  • Resizable text: Text can be resized up to 200% without loss of content or functionality

Operable

  • Keyboard accessible: All functionality is available from a keyboard
  • No keyboard traps: Focus can be moved away from any component using the keyboard
  • Skip navigation: Skip links are provided to bypass repetitive content
  • Focus visible: Keyboard focus indicator is always visible
  • Touch targets: Interactive elements have a minimum size of 44Ã—44 CSS pixels
  • No timing: There are no time limits on content

Understandable

  • Language: Page language is identified in HTML
  • Predictable: Navigation is consistent across pages
  • Input assistance: Form labels are clear and errors are identified
  • Consistent navigation: Navigation mechanisms are repeated consistently

Robust

  • Valid HTML: Pages use valid, well-formed HTML
  • Semantic markup: Proper use of headings, landmarks, and ARIA labels
  • Compatible: Content is compatible with current and future assistive technologies

Technologies Used

The accessibility of FlowTrack relies on the following technologies to work with the particular combination of web browser and any assistive technologies or plugins installed on your computer:

  • HTML5
  • WAI-ARIA
  • CSS
  • JavaScript

These technologies are relied upon for conformance with the accessibility standards used.

Known Limitations

Despite our best efforts to ensure accessibility of FlowTrack, there may be some limitations. Below is a description of known limitations:

  • Third-party content: Some external content (such as embedded social media feeds) may not be fully accessible. We are working with our partners to improve this.
  • Legacy content: Some older product images may lack optimal alternative text. We are in the process of reviewing and updating all content.

Compatibility with Assistive Technologies

FlowTrack is designed to be compatible with the following assistive technologies:

  • Screen readers (NVDA, JAWS, VoiceOver, TalkBack)
  • Screen magnification software
  • Speech recognition software
  • Alternative keyboards and input devices
  • Browser zoom functionality

FlowTrack is not compatible with browsers older than 3 major versions.
